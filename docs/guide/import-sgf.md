# Import and open SGF files (current behavior)

Gomoku does not yet support registering itself as the system default handler for `.sgf` files. To view a `.sgf` file in Gomoku, import it into the app and then open it from the Replays (review) list.

## How to import a `.sgf` file

- Open Gomoku and use the app's **Import** feature (for example: `File → Import` or the **Import** button) to select the `.sgf` file you want to load.
- Alternatively, use any import area or drag-and-drop support the app provides for game records.

## Open the imported game

- Go to the **Replays** (or Review) list inside Gomoku.
- Find the imported game and click to open it for review and playback.

## Notes

- Setting Gomoku as the default `.sgf` handler is not supported at this time.
- If you need system‑level default handling (double‑click to open), we can add this feature in a future release — please open an issue on the repository to request it.
